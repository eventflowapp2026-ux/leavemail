# 📧 LeaveMail - Smart Leave Request Assistant

LeaveMail is a minimalist, accessibility-focused mobile web application that simplifies sending academic leave request emails for students and parents. Built specifically for parents and guardians who may find typing long emails or navigating complex email applications difficult.

## ✨ Features

### Core Features
- 📅 **Smart Date Selection** - Easy date/time pickers with auto-duration calculation
- 🎨 **Multiple Email Templates** - Professional, Casual, Entertaining, and Custom templates
- 🎭 **Email Visual Themes** - Default, Modern, Elegant, Colorful, Minimal, Festive
- 🎤 **Voice Input** - Speak your leave reason instead of typing (Web Speech API)
- ⚡ **Quick Presets** - One-tap buttons for Sick, Family, Emergency, Medical leave
- 🔒 **Security Options** - PIN Lock (4-6 digits), Password Lock, or Pattern Lock (3x3 grid)
- 📋 **Leave History** - Track all sent requests locally with timestamps
- 📎 **Attachment Support** - Instructions for adding medical certificates/documents

### Accessibility Features
- 🌓 **High Contrast Mode** - Better visibility for visually impaired users
- 🔍 **Large Text Mode** - Increased font size for better readability
- 🎬 **Reduced Motion** - Disables animations for motion-sensitive users
- 🎤 **Voice Input** - Hands-free operation for users with limited mobility

### Visual Themes (Page Themes)
- 🎄 **Christmas Theme** - Festive red and green design
- 🌸 **Onam Theme** - Traditional Kerala festival colors
- 🎆 **New Year Theme** - Celebration style with purple accents
- 🇮🇳 **Independence Day Theme** - Tricolor design (orange, white, green)

## 🚀 Quick Start

### Prerequisites
- Python 3.7 or higher
- pip (Python package manager)
- Modern web browser (Chrome, Firefox, Edge, Safari)

### Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/leavemail.git
cd leavemail

# Install dependencies
pip install -r requirements.txt

# Run the application
python app.py
